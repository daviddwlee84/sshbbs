# 看板規範

歡迎來到 Welcome 看板。本板為新手綜合區，發文前請先閱讀以下規範。違反者由板主依下列等級處置；對處置有異議可寫站內信向站長申訴。

## 一、發文守則

1. **禁止謾罵 / 人身攻擊**：對事不對人。鎖定特定使用者批評、嘲諷、羞辱者一律刪文。
2. **禁止廣告 / 商業行為**：未經板主同意不得張貼商業連結、團購、推銷。
3. **禁止重複發文**：同主題請集中討論，避免洗版；尋找類似舊文可在標題前加 `Re:`。
4. **禁止散布不實資訊**：引用消息請附來源連結與出處；未經查證的爆料不在此板處理。
5. **發文標題請有意義**：避免「help」、「問一下」這種無資訊的標題；用 `[分類]` 前綴會更好。

## 二、推文守則

- 推文同樣適用上述發文守則。
- 不得連續推文洗版（同帳號連續 5 推以上視為洗版）。
- 推文僅作補充或回應之用，深度討論請另開新文。

## 三、違規處理

| 違規等級 | 處置 |
|---|---|
| 輕微 | 板主口頭提醒 / 刪推文 |
| 一般 | 刪文 + 站內信警告 |
| 嚴重 / 累犯 | 提報站長水桶（暫停發文權限） |

## 四、修訂紀錄

- **v1.0 (2026-05-06)** — 初版上線：定義發文 / 推文 / 違規處理三大區塊。

> 修訂條文採累積式：未來修訂請新增條目於本區（最新者列於頂端），保留歷史版本以供對照。
> 若條文有實質變更，請於 commit message / 站務公告同步說明變更原因。
